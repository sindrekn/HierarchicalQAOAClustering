# HierarchicalQAOAClustering

A quantum-classical hybrid pipeline for graph clustering using the Quantum Approximate Optimization Algorithm (QAOA). This repository implements a hierarchical bisection clustering framework using classical simulation of QAOA statevectors, alongside diagnostic analysis tools for parameter sweeps ($\gamma, \beta$, and circuit depth $p$) and visualization utilities.

---

## Pipeline Overview

The project consists of three core Python scripts designed to be executed sequentially or independently:
1. **`HierarchicalClusteringQAOA_ClassicalSimulation.py`**: The computational core. Runs the hierarchical clustering tree generation and/or statevector diagnostic sweeps.
2. **`TreeStructurePlot.py`**: Visualizes the hierarchical bisection tree structure generated by the simulation.
3. **`PlotClusters.py`**: Generates analytical plots such as parameter heatmaps, probability distributions across depths ($p$), and the final optimal graph partitions.

## Command-Line Arguments Reference

### 1. Classical Simulation & Optimization
`HierarchicalClusteringQAOA_ClassicalSimulation.py` handles both the tree building and the parameter diagnostic analysis.

| Argument | Type | Default | Description |
| :--- | :---: | :---: | :--- |
| **Mode Flags** | | | |
| `--hierarchical` | flag | `False` | Run the hierarchical $k$-cluster QAOA bisection tree generation. |
| `--analyse` | flag | `False` | Run $\gamma/\beta$ grid searches and $p$-sweep diagnostic analyses. |
| **Shared Parameters** | | | |
| `--n_restarts` | `int` | `5` | Number of random optimization restarts per QAOA execution. |
| `--gamma_max` | `float`| $4\pi$ | Maximum $\gamma$ value for random parameter initialization. |
| `--beta_max` | `float`| $\pi$ | Maximum $\beta$ value for random parameter initialization. |
| `--graphfile` | `str` | `"graphs/graph.csv"`| Path to the CSV file containing graph edges and weights. |
| `--alpha` | `float`| `1.0` | *[analyse mode]* Resolution parameter for diagnostic sweeps. |
| **Hierarchical-Only** | | | |
| `--hierarchical_result`| `str` | `"results/hierarchical_result.h5"` | Filename to save the output hierarchical clustering tree structure. |
| `--p` | `int` | `1` | QAOA circuit depth (number of layers). |
| `--min_cluster_size` | `int` | `2` | Minimum subgraph size required to attempt a further bisection split. |
| `--max_level` | `int` | `7` | Maximum allowable depth of the bisection tree. |
| `--alpha_scale` | `float`| `1.5` | Scaling factor applied to $\alpha$ at each successive depth level of the tree. |
| **Analyse-Only** | | | |
| `--analyse_results` | `str` | `"results/analyse_results.h5"` | Filename to save diagnostic sweep data. |

### 2. Tree Structure Visualization
`TreeStructurePlot.py` generates a tree diagram showcasing how the graph was split up level-by-level.

| Argument | Type | Required | Description |
| :--- | :---: | :---: | :--- |
| `--result_path` | `str` | **Yes** | Path to the generated hierarchical result `.h5` file. |
| `--save_path` | `str` | **Yes** | Destination path for the output tree diagram file (e.g., PDF/PNG). |

---

### 3. Clustering & Diagnostic Plotter
`PlotClusters.py` compiles metrics and optimization surfaces into visual layouts.

| Argument | Type | Default | Description |
| :--- | :---: | :---: | :--- |
| **Mode Selection** | | | |
| `--heatmap` | flag | `False` | Generate $\gamma$-$\beta$ cost landscape heatmaps. |
| `--optimal-config`| flag | `False` | Plot the final partitioned graph color-coded by cluster assignment. |
| `--prob-dist` | flag | `False` | Plot the output statevector probability distributions across depths $p$. |
| **File Paths** | | | |
| `--save_path` | `str` | `None` | Path where the generated figure will be saved. |
| `--analyse_path` | `str` | `None` | Path to the analysis HDF5 file (Required for `--heatmap` and `--prob-dist`). |
| `--hierarchical_path`| `str` | `None` | Path to the hierarchical HDF5 file (Required for `--optimal-config`). |
| `--graph_path` | `str` | `None` | Path to the source graph CSV file (Required for `--optimal-config`). |

## Example Usage

### Step 1: Run the QAOA Cluster Solver and Diagnostics
Execute both the hierarchical bisection routine and diagnostic analysis sweeps on your target graph:
```bash
python HierarchicalClusteringQAOA_ClassicalSimulation.py \
    --hierarchical \
    --analyse \
    --n_restarts 100 \
    --graphfile "graphs/XXSgraph.csv" \
    --hierarchical_result "results/XXSgraph-Hierarchical.h5" \
    --analyse_results "results/XXSgraph-analyse.h5"
